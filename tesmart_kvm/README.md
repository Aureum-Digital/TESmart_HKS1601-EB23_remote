# TESmart KVM Controller

Control a TESmart HKS1601-EB23 16-port HDMI KVM switch from Home Assistant:
web UI (via Ingress), MQTT entities with discovery, and a REST API.

- Switch between the 16 inputs with custom names and icons
- Mute/unmute the buzzer, set the LED timeout
- MQTT select/sensor/switch/button entities created automatically
- Auto-discovers the Mosquitto broker add-on

See DOCS for configuration details.
